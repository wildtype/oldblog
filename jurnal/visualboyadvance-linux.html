VisualBoyAdvance for Linux

28 August 2006

Beberapa hari ini saya jadi suka main GameBoy pake emulator VBA. Dulu nggak suka sama sekali, tapi karena ikut pembinaan utk OSN Komputer, jadi ketularan temen2 yang semuanya suka main GameBoy. Biasanya kalau sedang ditinggal pengajar atau udah judes mikir penyelesaian soal, pada main GB. Aku ikut2an aja main. Karena di rumah saya lebih sering pake Slackware, tadi siang iseng cari VBA untuk linux. Di websitenya VBA ketemu paket tarball untuk linux, ya udah download. Cuma 243an Kilobyte, tak pikir isinya source. Habis ekstrak, tak jalanin ‘make’ malah keluar ‘make: * No targets specified and no makefile found. Stop.’ . Lho? Setelah diliat ternyata isinya file binarynya VBA.

Ya sudah, file yang namanya VisualBoyAdvance dikopi ke /usr/bin terus dibuat softlink jadi vba. Waktu merun vba, malah keluar opsi yang banyak banget. Tak kiro koyo VisualBoyAdvance nggo Windows sing ana main menune ning main window :-). Ternyata harus masukkan nama file rom & opsi lewat terminal. Waktu coba main “One Piece”, kok cepet bgt? Throttle nya sampe 500%. Opsinya tak coba satu2, pake opsi -no-throttle nggak terlalu membantu. Kecepatannya turun tapi masih 200an persen.

Terus coba lagi, throttlenya 100%, malah lelet bgt, suarane ora karuan. Coba lagi throttlenya 150%, akhirnya pas.


Asiiik.
